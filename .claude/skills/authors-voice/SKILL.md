# authors-voice

This skill owns prose formation and prose audit only. It does not own dramatic design, canon, or lifecycle policy.

## Shared contract

Write clear, scene-first, phone-readable genre prose. The author stays invisible; literal action stays clear; close POV stays close; sincere emotion arrives through behaviour and choice; genre pleasure stays intact; ordinary language is allowed whenever it serves the scene.

## Composition mode

Load `references/compose.md` and the story-engine generation packet.

Do not load the rule registry while composing.

Do not load `references/audit.md` in composition mode.

## Audit mode

After a complete draft, start a fresh context and load `references/audit.md` to review the final prose.

If the audit finds material issues, repair only the accepted findings, then rerun audit on the materially changed passages.

If a finding is structural, route it to story-engine. If it is canon, knowledge, or spoiler handling, route it to serial-process. If it is about prose texture or line-level delivery, keep it here.
